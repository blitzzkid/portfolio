JumpStart Day 3

Day 3 was CSS day as we learned CSS in depth, understanding more about using relative units for responsive scaling as well us important layouts like grid and flexbox.

  • Rem takes reference from the root e.g. font-size: 16px;
  • Em takes reference from the font size defined in that tag
  • Use relative terms (em, rem) for spacing
  • Within a {}, put the font size at the top, for everything to take reference from the font size via em
  • BEM: Block Element Modifier to name classes
