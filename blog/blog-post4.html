JumpStart Day 4

This was mindblown day as we found out that when it comes to JavaScript, the more we know the more we realise we don't know! We learned the background of JavaScript, installing NodeJS, npm as well as basic JavaScript stuff such as assertions, variables, conditional statements and functions.

Npm:

  • Semantic versioning > 0.0.0 > Major ver. Minor ver. Patch
  • version - Never touch the package-lock.json
  • So if you see ~1.0.2 it means to install version 1.0.2 or the latest patch version such as 1.0.4. If you see ^1.0.2 it means to install version 1.0.2 or the latest minor or patch version such as 1.1.0.

JS:

  • Number++ increments the number after the statement
  • ++Number increments the number before the statement
  • Number += 1 is more readable
  • Number.isNaN(Number(“123”) === false - Means it is a number
  • && will be compared before ||
    https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Operators/Operator_Precedence
  • Global scope, function scope, block scope
  • Lexical scope - the concept describing how you can “see” variables outside of your scope, but not the opposite direction
  • IIFE: Immediate invoke function expression (Function)();
  • Rest parameters (…rest)
  • Function is a object, which can contain methods
  • Functions can return functions
  • Functions are treated as if they are moved to the top: Hoisting
  • Use === and !==
  • Use const unless youneed to change the value
  • Use arrow functions => {}
